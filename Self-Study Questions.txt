1. What is the difference between an adaptive website and a fully responsive website?
	
	* Adaptive website means that the there are several versions of each layout that change based on the screen size of the viewer.
	  Responsive website can change to different layouts depending on the screen size of the viewer.

2. Describe what it means to be mobile first vs desktop first.
	
	* Mobile first: is doing layout from mobile viewpoint which is focused more on functionality and adding more layers or elements to higher pixels screens.
	
	* Desktop first layout: is designedfor  higher resolution screens first so it contains a lot of elements. 
	  But as screen resolution lowers, some of the elements will be hidden, wrapped or modified to accomodate smaller screens
